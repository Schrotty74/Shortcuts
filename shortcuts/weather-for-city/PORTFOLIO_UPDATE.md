# Portfolio Update

Deutsch: [PORTFOLIO_UPDATE.de.md](PORTFOLIO_UPDATE.de.md)

## Short Description

`Weather for City` is a public Apple Shortcut that fetches current weather data
for a freely entered city and displays it as an animated, responsive weather
card.

## Portfolio Text

New public app: `Weather for City` is an Apple Shortcut for Mac and iPhone. It
uses public Open-Meteo APIs, requires no API key, and shows a bilingual
animated weather card with dynamic colors, weather-code scenes, and responsive
layout.

## GitHub Profile Text

New: `Weather for City` in the `Schrotty74/Shortcuts` repository.
